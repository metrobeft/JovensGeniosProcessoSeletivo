Na pasta contem 4 pastas listadas a seguir:

*JG programa;
*JG site;
*Logica Simples;
*Projeto;

Na pasta Logica Simples contém um arquivo em formato C com a lógica base que foi utilizada.

Na pasta JG site contém um site em HTML com Js e Bootstrap com uma interface agradável ao usuário para a execução do algoritmo.

Na pasta JG programa contém um programa em Unity no qual possui uma interface agradável ao usuário para a execução do algoritmo.

Na pasta Projeto contém o projeto em Unity antes de ser compilado.
